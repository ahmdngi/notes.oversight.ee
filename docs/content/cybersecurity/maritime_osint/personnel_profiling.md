# Category 3: Personnel Profiling (Human Factor Intelligence)

## Overview
Adversaries use AI to profile crew members for spear-phishing. This category focuses on identifying the digital footprint of crew and staff to assess social engineering risks.

## Social Media Analysis
1.  **[Huntintel](https://huntintel.io/)** - Discover social media posts from a specific area (e.g., a port).
2.  **[LinkedIn](https://www.linkedin.com/)** - Primary source for mapping maritime organizational structures and personnel.
3.  **[Maltego](https://www.maltego.com/)** - Link analysis tool for connecting people, companies, and assets.
4.  **[Social-Searcher](https://www.social-searcher.com/)** - Free social media search engine.
5.  **[Namechk](https://namechk.com/)** - Check username availability across platforms to identify user handles.
6.  **[Sherlock](https://github.com/sherlock-project/sherlock)** - Hunt down social media accounts by username.
7.  **[Pytster (GitHub)](https://github.com/GaneshBalaji-04/Pytster)** - Username enumeration tool.
8.  **[Wayback Machine](https://archive.org/)** - View previous versions of crew member profiles or deleted posts.

## Credential & Breach Checks
9.  **[Have I Been Pwned](https://haveibeenpwned.com/)** - Check if credentials have been exposed in data breaches.
10. **[DeHashed](https://www.dehashed.com/)** - Search engine for leaked credentials.
11. **[Intelligence X](https://intelx.io/)** - Search engine and data archive for leaks and public records.

## AI & Profiling Tools
12. **[FaceCheck.id](https://facecheck.id/)** - Reverse image search to verify identity using facial recognition.
13. **[PimEyes](https://pimeyes.com/)** - Advanced face recognition search engine.
