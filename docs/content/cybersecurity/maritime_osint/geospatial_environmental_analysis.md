# Category 4: Geospatial and Environmental Analysis

## Overview
Geospatial analysis provides physical context. It is vital for verifying AIS data (which can be spoofed) and understanding environmental conditions.

## Satellite Imagery
1.  **[Sentinel Hub](https://www.sentinel-hub.com/)** - Free access to Sentinel satellite imagery.
2.  **[Google Earth Pro](https://www.google.com/earth/versions/)** - 3D globe visualization and historical imagery.
3.  **[Zoom Earth](https://zoom.earth/)** - Real-time weather and satellite imagery.
4.  **[Planet](https://www.planet.com/)** - High-resolution satellite imagery (Commercial/Trial).
5.  **[Maxar](https://www.maxar.com/)** - High-resolution earth intelligence.

## Geolocation Tools
6.  **[Shadowmap](https://app.shadowmap.org/)** - 3D sunlight and shadow visualizer for verifying time of images.
7.  **[SunCalc](https://www.suncalc.org/)** - Sun position and phases for geolocation verification.
8.  **[TIB Geoestimation](https://labs.tib.eu/geoestimation/)** - AI-powered tool to estimate geographic location of an image.
9.  **[GeoGuessr](https://www.geoguessr.com/)** - Educational tool for practicing geolocation skills.
10. **[What3Words](https://what3words.com/)** - Precise location addressing system.

## Weather & Environment
11. **[Windy](https://www.windy.com/)** - Global weather forecast visualization.
12. **[NOAA Tides](https://tidesandcurrents.noaa.gov/)** - US tidal information.
13. **[Ocean Motion](https://www.oceanmotion.org/)** - Ocean surface currents.

## Webcam & Visual Analysis
14. **[WebcamTaxi](https://www.webcamtaxi.com/)** - Live webcams of ports and cities.
15. **[SkylineWebcams](https://www.skylinewebcams.com/)** - Live webcams worldwide.
16. **[Insecam](http://www.insecam.org/)** - Directory of live webcams (use ethically).

## Image Analysis Repos
17. **[Image Research OSINT (GitHub)](https://github.com/The-Osint-Toolbox/Image-Research-OSINT)** - Toolbox for image analysis.
18. **[Geolocation OSINT (GitHub)](https://github.com/The-Osint-Toolbox/Geolocation-OSINT)** - Tools for location intelligence.
