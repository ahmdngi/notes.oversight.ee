# Category 5: Digital Infrastructure Reconnaissance

## Overview
Maritime operations are increasingly digitized. This category focuses on the technical attack surface—exposed OT systems, port networks, and satellite communication links.

## Network Scanning & IoT Search Engines
1.  **[Shodan](https://www.shodan.io/)** - Primary tool for discovering exposed maritime OT/IT devices.
2.  **[Censys](https://search.censys.io/)** - Search engine for internet-connected devices.
3.  **[ZoomEye](https://www.zoomeye.org/)** - Cyberspace search engine.
4.  **[GreyNoise](https://www.greynoise.io/)** - Visualizer of internet background noise and scanning.
5.  **[BinaryEdge](https://www.binaryedge.io/)** - Scanning for exposed services.
6.  **[FOFA](https://fofa.info/)** - Cyberspace mapping.
7.  **[Threatseye Map](https://threatseye.io/threats-map)** - Real-time global map visualizing malware threats.

## Vulnerability & Threat Analysis
8.  **[NIST NVD](https://nvd.nist.gov/)** - National Vulnerability Database.
9.  **[CVE Mitre](https://cve.mitre.org/)** - List of Common Vulnerabilities and Exposures.
10. **[Exploit-DB](https://www.exploit-db.com/)** - Database of exploits and vulnerable software.
11. **[Vulners](https://vulners.com/)** - Vulnerability database and search engine.
12. **[Spyse](https://spyse.com/)** - Internet assets registry.

## Network Utilities
13. **[Robtex](https://www.robtex.com/)** - DNS research and IP info.
14. **[URLhaus](https://urlhaus.abuse.ch/)** - Database of malicious URLs.
15. **[URLScan](https://urlscan.io/)** - Scan and analyze websites.

## OSINT Frameworks
16. **[OSINT Framework](https://osintframework.com/)** - Interactive diagram of OSINT tools.
17. **[OSINT-TOOLS-CLI (GitHub)](https://github.com/Coordinate-Cat/OSINT-TOOLS-CLI)** - Command-line interface for OSINT tools.
