# Category 6: Analytical and Threat Assessment Tools

## Overview
This category encompasses the frameworks and repositories used to synthesize raw data into actionable intelligence. It supports the "Analysis" phase of the intelligence cycle.

## Threat Intelligence Platforms
1.  **[MISP](https://www.misp-project.org/)** - Open source threat intelligence platform.
2.  **[OpenCTI](https://www.opencti.io/)** - Open Cyber Threat Intelligence platform.
3.  **[MITRE ATT&CK Navigator](https://mitre-attack.github.io/attack-navigator/)** - Visualizing adversary techniques.
4.  **[ThreatConnect](https://threatconnect.com/)** - Threat intelligence operations platform.
5.  **[YARA](https://virustotal.github.io/yara/)** - Tool for identifying and classifying malware.
