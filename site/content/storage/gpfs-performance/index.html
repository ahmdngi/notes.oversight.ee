    HPC keskus : GPFS Performance testing⚓︎

    Performance testing needs to be done to gauge the storage clusters performance over different use cases.

    Copying with dd or rsync doesn't give a real overview of performance.

    Link to FIO testing: https://fio.readthedocs.io/en/latest/fio_doc.html
